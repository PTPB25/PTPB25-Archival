# IF2130 — Sistem Operasi

<p align="justify">
Mata kuliah Sistem Operasi mempelajari fungsi dan peran sistem operasi dalam mengelola sumber daya komputer, penggunaan lanjutan sistem operasi, serta konsep dan prinsip desainnya. Mata kuliah ini juga melatih analisis berbagai desain sistem operasi dan implementasi dari modul-modul sederhana Sistem Operasi.
</p>
