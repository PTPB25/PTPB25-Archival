# IF2150 — Rekayasa Perangkat Lunak

<p align="justify">
Mata kuliah Rekayasa Perangkat Lunak membahas prinsip, metode, dan proses pengembangan perangkat lunak secara terstruktur mulai dari analisis kebutuhan, perancangan sistem, implementasi, pengujian, hingga pemeliharaan perangkat lunak. Mata kuliah ini lebih terfokus ke dalam proses dokumentasi dalam pengembangan perangkat lunak dibandingkan proses implementasinya secara langsung. 
</p>

| **Bobot akademik** |
| :--- |
| 1 Tugas Besar **(10 Milestone)** |
| Ujian Tengah Semester (UTS) |
| Ujian Akhir Semester (UAS) |

## Tugas Besar

<p align="justify">
Tugas Besar <strong>IF2150 - Rekayasa Perangkat Lunak</strong> merupakan proyek pembuatan <strong>perangkat lunak berbasis SDG</strong>. Tubes ini dimulai pada minggu kedua dan memiliki <i>milestone</i> setiap minggunya sampai minggu terakhir sebelum UAS. <i>Deliverable</i> dari tubes ini adalah sebuah perangkat lunak fungsional beserta dengan seluruh dokumentasi pembuatannya
</p>
