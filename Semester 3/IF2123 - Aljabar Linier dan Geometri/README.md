# IF2123 — Aljabar Linier dan Geometri

<p align="justify">
Mata kuliah Aljabar Linier dan Geometri mempelajari konsep dasar aljabar linier seperti matriks, sistem persamaan linier, determinan, vektor, transformasi linier, serta penerapannya. Mata kuliah ini termasuk dalam rumpun Lab IRK. 
</p>

| **Bobot akademik** |
| :--- |
| 2 Tugas Besar |
| 1 Makalah |
| Ujian Tengah Semester (UTS) |
| Ujian Akhir Semester (UAS) |

Referensi mata kuliah: <code>[Website Rinaldi Munir 2025](https://informatika.stei.itb.ac.id/~rinaldi.munir/AljabarGeometri/2025-2026/algeo25-26.htm)</code>

## Tugas Besar 1

<p align="justify">
Tugas Besar 1 IF2123 — Aljabar Linier dan Geometri merupakan proyek pembuatan <b>pustaka perhitungan matriks</b> dalam bahasa <b>Java</b>. Tubes ini dikerjakan secara berkelompok dengan tiap kelompok berjumlah tiga orang. <b>Tubes</b> ini menghasilkan tiga luaran utama: <b>pustaka Java</b> berbentuk <code>.jar</code> untuk operasi matriks, regresi, dan interpolasi; <b>aplikasi CLI atau GUI</b> yang menggunakan pustaka tersebut dengan dukungan input keyboard maupun berkas <code>.txt</code> serta dokumentasi <code>README</code> di GitHub; dan <b>laporan</b> yang menjelaskan implementasi pustaka dan program.
</p>

Spesifikasi resmi tugas besar:   **<code>[Dokumen Spesifikasi Tugas Besar 1](https://informatika.stei.itb.ac.id/~rinaldi.munir/AljabarGeometri/2025-2026/Tubes1-IF2123-Algeo-2025.pdf)</code>**

## Tugas Besar 2

<p align="justify">
Tugas Besar 2 IF2123 — Aljabar Linier dan Geometri merupakan proyek pembuatan <b>aplikasi E-Library berbasis web</b>. Serupa dengan tubes 1, tubes ini dikerjakan secara berkelompok dengan tiap kelompok berjumlah tiga orang (harus berbeda dari kelompok tubes 1). Aplikasi yang dibangun wajib menggunakan framework frontend berbasis <b>JavaScript/TypeScript</b> dengan backend mengandung bahasa pemrograman yang mendukung modul operasi matriks <b>(Python, Go, Rust, etc)</b>.
</p>

Spesifikasi resmi tugas besar:   **<code>[Dokumen Spesifikasi Tugas Besar 2](https://s.hmif.dev/algeo2-25)</code>**
